# MetaPohyb

> Smart physiotherapy & exercise tracking — connecting clients and physiotherapists through guided exercises, VR device integration, and real-time progress monitoring.

## What is MetaPohyb?

MetaPohyb is an open-source mobile application built with Flutter. It bridges the gap between physiotherapists and their clients by providing:

- **Guided exercise sessions** — structured routines designed and assigned by physiotherapists
- **VR device integration** — connect VR glasses over local WiFi for immersive rehabilitation
- **Client & Physio roles** — dedicated dashboards for both sides of the therapy workflow
- **Session history & progress tracking** — every session is recorded and reviewable
- **Secure cloud sync** — powered by Supabase with email verification
